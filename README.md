# Whitelist-VPN-Russia
Приватный репозиторий проекта Ą̷Ꮑ̸ƅ𐐆-ᐯᖘᚻ для подписки на анонимность и полную приватность в сети и пробив сетевых ограничений. Доступ только по приглашению если ты здесь, значит в доверенной группе, списка людей.

• Проект заточен именно под жёсткие российские ограничения:
эффективный обход черных списков РКН (реестр блокировок, DPI по доменам/IP/протоколам/сигнатурам);

• полный пробив белых списков операторов (когда мобильный интернет пропускает только "разрешённые" ресурсы — госуслуги, банки, Яндекс, ВК, Озон и т.п., а всё остальное режется DPI на уровне провайдера);

• устойчивость к глушилкам, фильтрам протоколов и активному противодействию цензуре.
Проверенно работает на мобильных сетях МТС, T2 (Tele2), МегаФон, Yota, Билайн и других популярных операторах — даже при самых жёстких белых списках и отключениях всего лишнего.

Реализация использует обфускацию, маскировку трафика и адаптацию под DPI для максимальной стабильности и минимизации отслеживания. Всё исключительно в закрытой экосистеме — без публичного распространения, рекламы или открытого использования.
