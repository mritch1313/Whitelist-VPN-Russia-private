# Whitelist-VPN-Russia
Приватный репозиторий проекта Ą̷Ꮑ̸ƅ𐐆-ᐯᖘᚻ для подписки на анонимность и полную приватность в сети и пробив сетевых ограничений. Доступ только по приглашению если ты здесь, значит в доверенной группе, списка людей.

• Проект заточен именно под жёсткие российские ограничения, цензуры, ТСПУ-DPI: эффективных обходов черный списков (black list), белого списка (white list), глушилок, запретов, сетевых-ограничении, обход-цензуры РКН (реестр блокировок, DPI по доменам/IP/протоколам/сигнатурам/шум/трафика);

• полный пробив белых списков, черных списков различных операторов/сервисов (когда мобильный интернет заблокирован, и пропускает только "разрешённые" ресурсы — госуслуги, банки, Яндекс, ВК, Озон и т.д а всё остальное режется DPI на уровне провайдера/серверов);

• устойчивость к глушилкам, фильтрам протоколов и активному противодействию цензуре.
Проверенно работает на мобильных сетях МТС, T2 (Tele2), МегаФон, Yota, Билайн и других популярных операторах даже при самых жёстких белых списках и отключениях всего интернета.

Реализация использует обфускацию, маскировку трафика разные протоколы и адаптацию под DPI для максимальной стабильности и минимизации отслеживания/треккеров. Всё исключительно в закрытой экосистеме — без публичного распространения, рекламы или открытого использования.
